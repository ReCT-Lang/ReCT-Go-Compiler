# ReCT Built-ins
This package contains any datatypes or functions that are built right into ReCT.  
These are things like:  
  
**Datatypes**: `void, bool, int, float, string`  
  
**Functions**:
- `Print()`
- `Write()`
- `Input()`
- `InputKey()`
- `Clear()`
- `SetCursor()`
- `GetSizeX()`
- `GetSizeY()`
- `SetSize()`
- `SetCursorVisible()`
- `GetCursorVisible()`
- `Random()`
- `Sleep()`
- `Version()`
